## 3️⃣ Financial Loan Analytics Dashboard

**Domain:** Banking & Finance

An executive-level financial analytics dashboard built using a real-world loan portfolio dataset to help financial institutions monitor lending performance, borrower behavior, repayment trends, and portfolio risk.

The project follows a complete Business Intelligence workflow, including data cleaning, ETL, star schema data modeling, DAX-based KPI development, and interactive dashboard design.

### Dashboard Pages

#### 📄 Executive Summary

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income (DTI)
* Good vs Bad Loan KPIs
* Monthly Loan Application Trends
* Loan Status Distribution

#### 📄 Loan Portfolio Analysis

* Loan Purpose Analysis
* Home Ownership Distribution
* Employment Length Analysis
* State-wise Loan Distribution
* Term-wise Loan Analysis
* Grade & Sub-Grade Performance
* Funding vs Amount Received
* Interactive Drill-down Analysis

#### 📄 Credit Risk Analysis

* Charged-Off Loan Analysis
* Good vs Bad Loan Ratio
* Interest Rate Risk Distribution
* DTI Risk Analysis
* Loan Performance by Grade
* Portfolio Risk Monitoring
* Interactive Risk Filters

### Business Insights

* Evaluated overall portfolio health using lending and repayment KPIs.
* Identified borrower segments contributing to higher default risk.
* Analyzed repayment performance across loan grades and purposes.
* Compared funded amounts with recovered payments to assess portfolio efficiency.
* Enabled decision-makers to monitor lending performance through interactive dashboards.

### Skills Applied

* Star Schema Data Modeling
* Power Query ETL
* Advanced DAX
* Financial KPI Development
* Data Cleaning & Transformation
* Business Intelligence Reporting
* Interactive Dashboard Design
* Financial Data Analysis
